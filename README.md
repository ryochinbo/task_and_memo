# 看板方式タスク管理ツール

ドラッグ&ドロップで操作できるシンプルな看板方式（カンバン方式）のタスク管理ツールです。

## 特徴

- **看板方式のインターフェース**: TODO、進行中、完了の3つのカラムでタスクを管理
- **ドラッグ&ドロップ**: 直感的な操作でタスクの状態を変更
- **豊富なタスク情報**: メモ、重要度、タグ、締め切りなど
- **ボール管理機能**: 誰が作業中か（ボールを持っているか）を可視化
- **自動Git保存**: タスク変更時に自動でgit commit & push

## 動作環境

- Windows 10/11
- Python 3.11

## インストール方法

1. リポジトリをクローンまたはダウンロード

2. `start.bat` をダブルクリックして実行
   - 仮想環境（`.venv`）が自動作成されます
   - 必要なパッケージ（Flask）が自動インストールされます

3. 自動的にブラウザが開きます
   - URL: http://127.0.0.1:5000

## 使い方

### タスクの作成

1. 「+ 新規タスク」ボタンをクリック
2. 以下の情報を入力：
   - **タスク名**: 必須
   - **状態**: TODO / 進行中 / 完了
   - **重要度**: 高 / 中 / 低
   - **メモ**: タスクの詳細説明
   - **タグ**: カンマ区切りで複数入力可能
   - **タグ自由記述**: 自由形式のタグ情報
   - **ボールが自分にある**: 作業中かどうかのフラグ
   - **ボール保持者名**: ボールが自分にない場合の担当者名
   - **開始日**: タスク開始日
   - **締め切り**: タスク期限

### タスクの状態変更

- タスクカードをドラッグして、別のカラムにドロップ
- 自動的に保存されます

### タスクの編集・削除

- タスクカードの「編集」ボタンで編集
- 「削除」ボタンで削除（確認ダイアログあり）

## ファイル構成

```
task_and_memo/
├── app.py              # Flask バックエンドサーバー
├── requirements.txt    # Python依存パッケージ
├── start.bat           # 起動バッチファイル
├── templates/
│   └── index.html      # HTMLテンプレート
├── static/
│   ├── style.css       # スタイルシート
│   └── app.js          # フロントエンドJavaScript
├── data/
│   └── tasks.json      # タスクデータ（Git管理）
└── docs/
    └── spec.md         # 仕様書
```

## データ保存

- タスクデータは `data/tasks.json` にJSON形式で保存されます
- 変更時に自動的に `data` ディレクトリでGitコミット＆プッシュが実行されます

## 終了方法

- コマンドプロンプトで `Ctrl + C` を押してください
